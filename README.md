## Getting Started

### Installation

Install dependencies:
```bash
npm install
```

### Environment Variables

Create a `.env.local` file:
```env
NEXT_PUBLIC_BACKEND_URL=http://localhost:3000
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

### Build

```bash
npm run build
npm start
```
